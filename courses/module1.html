The Trinity Pillars

To build for the web, you must master the three pillars of frontend development: Structure, Style, and Logic.

1. HTML (HyperText Markup Language)

Definition

HTML is the Structure. It acts as the "bones" or skeleton of a website, defining the semantic meaning of every element on the page, such as headings, paragraphs, and links. 

<!-- index.html -->
<article>
  <h1>The Main Heading</h1>
  <button class="cta-btn">Get Started</button>
</article>

2. CSS (Cascading Style Sheets)

Definition

CSS is the Presentation. It acts as the "skin" or visual layer. It describes how HTML elements are to be displayed on screen, including layout, colors, and fonts. 

/* style.css */
.cta-btn {
  background: #6366f1;
  color: white;
  border-radius: 8px;
  padding: 12px 24px;
}

3. JavaScript (JS)

Definition

JavaScript is the Behavior. It acts as the "nerves" or brain. It allows you to create interactive, dynamic content that responds to user actions like clicks and keyboard input. 

// script.js
const btn = document.querySelector('.cta-btn');

btn.addEventListener('click', () => {
  alert('Welcome to the Web!');
});

Interface Level

The DOM Tree

The Document Object Model (DOM) is the bridge that allows JavaScript to "talk" to your HTML. It transforms your static document into a dynamic interface.

1. What is the DOM?

Definition

The DOM is a programming interface for web documents. It represents the page so that programs can change the document structure, style, and content. The DOM represents the document as nodes and objects. 

// The Browser turns HTML into a "Tree"
Window
 └── Document
      └── <body>
           └── <div>
                └── <h1>Text Content</h1>

2. Selectors (Finding Elements)

Definition

To change something on the page, you must first "select" it. We use Selectors to target specific HTML elements. querySelector is the modern standard because it uses CSS syntax. 

// Selecting by ID or Class
const header = document.querySelector('#main-title');
const items = document.querySelectorAll('.list-item');

3. Event Listeners (User Interaction)

Definition

An Event Listener is a procedure in JavaScript that waits for an event to occur (like a click, a scroll, or a keypress). When the event triggers, the listener executes a function. 

// Listening for a user click
button.addEventListener('click', () => {
  console.log("The user interacted with the UI!");
});

JavaScript Logic

If HTML is the bones and CSS is the skin, JavaScript is the brain. It controls how data moves and how the application thinks.

1. Variables (Data Storage)

Definition

A Variable is a container for storing data values. In modern JS, we use const for values that never change and let for values that will be updated. 

// Storing data
const price = 99; 
let score = 0;

// Updating data
score = score + 10;

2. Functions (Action Blocks)

Definition

A Function is a block of code designed to perform a particular task. It is executed when "something" invokes or calls it. Modern developers use Arrow Functions for cleaner code. 

// Defining an action
const greetUser = (name) => {
  return "Hello, " + name;
};

// Running the action
console.log(greetUser("Alex"));

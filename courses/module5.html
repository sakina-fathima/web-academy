Optimization

Writing code that works is the first step. Writing code that is efficient, fast, and scalable is what defines a Senior Engineer.

1. Code Complexity (Big O)

Definition

Big O Notation measures how time or space requirements grow as data increases. It's the metric for code efficiency. 

// O(1) - Constant Time (Instant)
const val = arr[0];

// O(n) - Linear Time (Grows with size)
arr.forEach(i => console.log(i));

2. Debouncing

Definition

Debouncing limits the rate at which a function fires. Crucial for search inputs and scroll events to prevent performance lag. 

let timer;
input.oninput = () => {
  clearTimeout(timer);
  timer = setTimeout(() => search(), 500);
};
